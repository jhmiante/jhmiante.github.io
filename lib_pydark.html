Descrição

A Biblioteca "pydark" foi desenvolvida pensando em adicionar funcionalidades para facilitar o desenvolvimento, essa biblioteca não é específica para um fim, mas sim para diversos, criando assim facilitadores, segue abaixo a documentação completa da bilblioteca

Instalação

pip install pydark

Importação

import pydark as pk

Módulos

  • pydark.date
    • pydark.date.dtget

pydark.date

O módulo date serve para facilitar a manipulação de datas, sejam cálculo ou recuperar dados referente a datas

pydark.date.dtget

O módulo dtget serve para facilitar a recuperação de dados referentes a datas, como último dia do mês ou verificar quantos dias úteis possui o mês

pydark.date.dtget.getUtilDay

pydark.date.dtget.getUtilDay( dt, util_day, disregard=[], last_day=False )

A Função getUtilDay serve para buscar o último dia útil do mês de uma data informada, os dias úteis são considerados de Segunda a Sexta, podendo ser informado por parâmetro quais são os dias para desconsiderar como útil (podendo ser feriado, ou alguma condição específica para o cálculo)

Parâmetros:
dt: Objeto Datetime
util_day: Objeto Integer - Qual o dia útil que quer receber no retorno
disregard: Objeto List Integer - Lista de Dias que quer desconsiderar como útil
last_day: Objeto Boolean - Default False - Se quer travar dentro do mês, caso o dia ultrapasse o mês

Exemplo:


from datetime import datetime
from pydark.date.dtget import getUtilDay

dia = datetime.now()
dia_util = getUtilDay(dia, 5)
print(dia_util)
